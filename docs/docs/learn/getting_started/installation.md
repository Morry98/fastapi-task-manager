# Installation

## Install FastAPI Task Manager

The first step is to install FastAPI Task Manager.

Make sure you have set up a [FastAPI project](https://fastapi.tiangolo.com/#installation){.internal-link target=_blank} and then **install FastAPI Task Manager**:

<!-- termynal -->
```console
$ pip install fastapi-task-manager

---> 100%
Successfully installed fastapi-task-manager
```
